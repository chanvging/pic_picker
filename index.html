Welcome to use Pic Picker.

Pic Picker is an extension for google chrome, It will get the BIG pictures in the page and the next pages, and display them as scroll.

Usage

There are two ways to tragger the extenstion:

  • click the icon displyed in the toolbar of chrome
  • right click a link, choose "pick it" in the context menu

The first method will get the pictures in the current document, while the second method will handle the link on which context menu popup
